На борту у модуля usb разъём, при подключении к raspberry, он появляется в устройствах /dev/cdc-wdm0
sudo apt-get install libqmi-utils udhcpc
sudo qmicli -d /dev/cdc-wdm0 —dms-set-operating-mode='online'
